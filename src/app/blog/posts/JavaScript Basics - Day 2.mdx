## Comparison Operators

Comparison operators are used to compare values and return `true` or `false`.

<CodeBlock
    marginBottom="16"
    highlight="5"
    codes={[
  {
    code:
`let x = 5;
let y = "5";

console.log(x == y);  // true  (loose equality, only checks value)
console.log(x === y); // false (strict equality, checks value + type)
console.log(x != y);  // false
console.log(x > 3);   // true
console.log(x <= 5);  // true`,
    language: "js",
    label: "comparison.js"
  }
]} />

## Logical Operators

Logical operators combine conditions.

<CodeBlock
    marginBottom="16"
    highlight="4"
    codes={[
  {
    code:
`let isLoggedIn = true;
let isAdmin = false;

console.log(isLoggedIn && isAdmin); // false (AND)
console.log(isLoggedIn || isAdmin); // true  (OR)
console.log(!isLoggedIn);           // false (NOT)`,
    language: "js",
    label: "logical.js"
  }
]} />

## Assignment Operators

## Expressions

An **expression** is any valid code that produces a value.

## Why This Matters

Operators and expressions are the **building blocks of logic** in JavaScript.  
They allow you to calculate values, compare results, and control program flow.
